pandas==0.24.1

Pillow==5.3.0

scikit-learn==0.21.2

# How to run 

For reproducint results of GraphMix(GCN) of Table1 in the paper, go to directory GraphMix/semisupervised/codes and run the following commands:

`python run_cora.py`

`python run_citeseer.py`

`python run_pubmed.py`
